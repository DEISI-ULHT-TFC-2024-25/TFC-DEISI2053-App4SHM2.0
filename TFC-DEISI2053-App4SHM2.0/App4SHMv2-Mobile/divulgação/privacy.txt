PRIVACY POLICY FOR MOBILE APPLICATION "App4SHM"
-----------------------------------------------
Last updated: July 25, 2023

This privacy policy governs your use of the software application “App4SHM” (later referred as the/this “Application”) for Android™ mobile devices that was created by the Civil Research Group and DEISI, funded by COFAC.

The Application is is a smartphone application for structural health monitoring (SHM) of bridges or other civil structures to assess their condition after a catastrophic event or when required by authorities and stakeholders.

-----------------------------------------------
- What information does the Application obtain and how is it used?

The Application does not require any user registration, although some users may be registered outside the Application. The Application does not require the user to provide any personal information.

The Application may be used as guest or as an authenticated user. In the latter case, the Application obtains the userid and password, to authenticate the user in the server and all the actions performed in the Application are associated with that userid.

-----------------------------------------------
The Application collects:

- The id of the structure that the user intends to monitor
- The accelerometer data explicitly captured by the user, while monitoring the structure
- The 3 main frequencies within the power spectrum that is associated with accelerometer data

This information is sent and stored in the app4shm server. If the user is authenticated, the collected information will be associated with his userid.

-----------------------------------------------
The Application does not collect any information automatically.

The Application does not share any information automatically with third parties.

The Application does not collect precise information about the location of your mobile device.
