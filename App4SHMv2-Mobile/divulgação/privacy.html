PRIVACY POLICY FOR MOBILE APPLICATION "App4SHM"

Last updated: July 24, 2023

This privacy policy governs your use of the software application “App4SHM” (later referred as the/this “Application”) for Android™ mobile devices that was created by the Civil Research Group and DEISI, funded by COFAC.

The Application is is a smartphone application for structural health monitoring (SHM) of bridges or other civil structures to assess their condition after a catastrophic event or when required by authorities and stakeholders.

What information does the Application obtain and how is it used?

User Provided Information

The Application does not require any user registration, some users may be registered outside the Application. The Application does not require the user to provide any personal information.

The Application may be used as guest or as an authenticated user. In the latter case, the Application obtains the userid and password, to authenticate the user in the server and all the actions performed in the Application are associated with that userid. The userid can be any sequence of numbers and digits and can't be associated with a particular real person.

The Application collects:

  • The id of the structure that the user intends to monitor
  • The accelerometer data explicitly captured by the user, while monitoring the structure
  • The 3 main frequencies within the power spectrum that is associated with accelerometer data
This information is sent and stored in the app4shm server. If the user is authenticated, the collected information will be associated with his userid.

Automatically Collected Information

The Application does not collect any information automatically.

The Application does not share any information automatically with third parties.

The Application does not collect precise information about the location of your mobile device.

Security

We are concerned about safeguarding the confidentiality of your information stored locally on your mobile device.

Please be aware that, although we endeavor provide reasonable security for information we process and maintain, no security system can prevent all potential security breaches (e.g., a zero-day attack targeting mobile Operating System – OS).

In order to ensure an optimal security, you shall regularly install any Application’s update available from the mobile application marketplace.

We also advise you to regularly check for any Android OS update(s) that could be available and to enable automatic updates.
